## 📚 프로젝트 소개
> 문서 간의 관계를 보여주는 노트 서비스입니다.

노션과 옵시디언과 같이 사용자에게 편리한 메모 서비스를 제공합니다. <br/>
더 나은 코드를 위한 디자인 패턴에 대해 집중했습니다.

[API Docs](https://devb33.github.io/metro-server/src/main/resources/docs/index.html)

<br>

## 🔖 기능

### 소셜 로그인
- 네이버, 카카오, 구글 OAuth2.0을 활용한 로그인이 가능합니다.
- Spring Security를 사용하여 JWT 기반 인증 및 인가를 합니다.

<br/>

## 🏛️ 아키텍처

### 기술 스택
**Language** | Java 17

**Framework** | Spring Boot, Spring Data JPA

**Database** | H2

**Build Tool** | Gradle 8.8

**Deploy** | Docker
